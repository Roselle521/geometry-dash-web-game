# 几何冲刺

《几何冲刺》是一个竖屏微信小游戏 v1 原型：三跑道滑动换道、关卡跑酷、关末 Boss、本地存档、皮肤商店和基础音效反馈。

## 运行方式

- 微信小游戏：用微信开发者工具导入本目录，编译类型为小游戏，入口文件为 `game.js`。
- 浏览器预览：在本目录启动静态服务器后访问 `index.html`，用于本地调试核心玩法。
- 本地检查：运行 `npm test` 验证核心状态机、碰撞、胜负、商店和存档逻辑。

## v1 范围

- 100 个关卡，每关普通跑酷段为 2 分钟，难度按速度、障碍密度、Boss 血量和攻击频率递增。
- 本地单机存档，key 为 `geometry-dash-save-v1`。
- 结算时会把本局收集金币写入存档；通关时额外获得关卡奖励。
- 商店支持金币解锁皮肤，皮肤提供少量生命或金币加成。
- 不包含广告、排行榜、登录、云存档或分享。
